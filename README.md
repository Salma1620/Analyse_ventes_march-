## Description des Colonnes du Dataset du système de vente.

### Colonnes :

**ID_LIGNE** : Identifiant unique de chaque ligne de commande.
- **ID_COMMANDE** : Identifiant unique de la commande (peut contenir plusieurs lignes).
- **nom client** : Nom du client ayant passé la commande.
- **segment** : Segment du client (ex. *Consumer*, *Corporate*, *Home Office*).
- **region** : Région géographique du client.
- **pays** : Pays du client.
- **ville** : Ville du client.
- **zone reographique** : Zone géographique plus large, regroupant plusieurs régions (ex. *Europe*, *Amérique du Nord*).
- **ID_PRODUIT** : Identifiant unique du produit commandé.
- **nomproduit** : Nom du produit.
- **sous categorie** : Sous-catégorie du produit (ex. *Chaises*, *Téléphones*).
- **categorie** : Catégorie générale du produit (ex. *Mobilier*, *Technologie*).
- **DATE_COMMANDE** : Date à laquelle la commande a été passée.
- **DATE_EXPEDITION** : Date à laquelle la commande a été expédiée.
- **MODE_EXPEDITION** : Méthode d’expédition utilisée (ex. *Standard Class*, *Express*).
- **QUANTITE** : Nombre d’unités commandées du produit.
- **REMISE** : Réduction appliquée sur le prix (en pourcentage ou valeur).
- **PROFIT** : Profit généré par la vente de cette ligne.
- **MONTANT_VENTES** : Montant total de la vente (après remise).
